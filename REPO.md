# Rogue Portfolio Core

Full-stack portfolio application with backend API and frontend Next.js app.

## Structure

```
/backend          - NestJS backend (root dir deployment)
/app-web          - Next.js frontend (root dir deployment)
```

## Deployment

**Backend:** Set Railway root dir to `backend`
**Frontend:** Set Railway root dir to `app-web`

Each service has its own `nixpacks.toml` and `railway.json` for independent deployment.
